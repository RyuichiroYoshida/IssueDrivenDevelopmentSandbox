# IssueDrivenDevelopmentSandbox

## プロダクトの概要

GitHubのコミット履歴(芝)をユーザープロフィールページへアクセス無しで、一括で確認出来るようにし、

コミット履歴に応じたリアクションを行うプロダクト

## プロダクトの目的

GitHubへのコミット履歴を通じて作業量を可視化し、

継続的に作業を行っている人間や、一定の期間で多くの作業を行っている者を表彰することで、

モチベーション向上へと繋げる

## 対象ユーザー

Git(GitHub)を利用しているプログラマー

### 非対称ユーザー

Git(GitHub)を利用していない人間
